# Pardus Ağ Dosya Paylaşımı
Pardus için yerel ağ üzerinden QR kod ile dosya paylaşım sistemi.

# Kurulum Adımları
1. Konsola "cd Masaüstü" komudunu girin.
2. Konsola "git clone https://github.com/ayazyigit/pardus-ag-dosya-paylasimi" komudunu girin.
3. Masaüstünde yeni açılan "pardus-ag-dosya-paylasimi" klasörünün içindeki "pardus-ag-dosya-paylasimi.deb" dosyasını çalıştırın.
4. Çıkan ekranda bulunan "Yükle" tuşuna basın.
5. Yüklenme tamamlandıktan sonra başlatma menüsünden "Pardus Share" uygulamasını aratarak uygulamayı çalıştırın.

# Kullanma Talimatları
1. Uygulamadaki "Dosyayı Gözat..." butonuna basınız.
2. Açılan pencerede istediğiniz dosyayı seçiniz ve "Open" tuşuna basınız.
3. "Paylaşımı Başlat" tuşuna basınız.
4. Sağ tarafta bulunan QR kodunu telefonunuzdan okutunuz.

# Kaldırma Adımları
1. Pardus Yazılım Merkezi'ne girin.
2. Üst taraftan "Depo" bölümüne giriş yapın.
3. Arama kısmına "pardus-share" yazın.
4. Çıkan ilk depoya basın ve sağ tarafta çıkan kırmızı "Kaldır" butonuna basın.
